## 🔄 版本说明

### 版本号规则
采用语义化版本控制 (SemVer)：
- **主版本号**: 不兼容的API修改
- **次版本号**: 向下兼容的功能性新增
- **修订号**: 向下兼容的问题修正

### 更新类型
- 🎉 **新增功能** - 全新的功能特性
- 🔧 **功能改进** - 现有功能的增强
- 🐛 **问题修复** - 错误和缺陷的修复
- 📚 **文档更新** - 文档的修改和完善
- 🔒 **安全修复** - 安全相关的修复
- ⚡ **性能优化** - 性能改进和优化

### 支持政策
- **当前版本**: 完全支持，持续更新
- **前一版本**: 安全修复和重要缺陷修复
- **更早版本**: 仅提供安全修复
